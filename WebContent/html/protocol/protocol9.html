Validation Protocol

Test Description:
The purpose of this protocol is validation of Dose Initiation Calculation outputs. The Dose Initiation will be tested for algorithms, inputs and outputs, rounding to one and two decimals, as applicable. Tester will manually calculate doses using the same algorithms, values and rounding rules, compare manual calculations results with Dose Initiation Calculator’s outputs and verify that the Dose Initiation calculators algorithms and rounding rules function as expected.

Acceptance Criteria:
Acceptance criteria are defined as successful completion of the steps identified below with results matching the description of Expected Results.

Notes to Tester:
  • The “Comments” field should be left blank unless the step requires additional explanation
  • N/A = (not applicable) N/A in each step is used for navigational steps.
  • Tester should use standard MS Calculator installed on tester’s workstation in scientific mode for required manual calculations.
  • To set MS Calculator in scientific mode:
    • Select View on the menu and then Scientific mode. It will display zeros after the decimal point.
  • Rounding is a method used to shorten numbers. Rounding involves Rounding Up and Rounding Down.
    • Rounding up -If the number to the right of the rounding digit is 5 or greater, then the rounding digit is increased by one number and the number to the right is truncated.
    • Rounding Down - If the number to the right of the rounding digit is less than 5, then the rounding digit remains the same and the number to the right is truncated.
  • Most current version of US PI available from www.rxabbvie.com/pdf/zemplarivpi.pdf. Document number available at end of document, e.g. EN-2945 January, 2013.
