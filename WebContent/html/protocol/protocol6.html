Validation Protocol

Test Description:
The purpose of this protocol is to demonstrate with ZemPro that the Dose Titration Calculator displays warning messages for input values or Ca x P product out of range, prevents results calculations when displaying warning messages caused by User inputs, and accepts and displays values per the required numerical format. The protocol will also demonstrate the ability to view additional information on receiving a warning message and that the user is directed to the Warnings, Clinical Studies, Dosage and Administration sections within the US Prescribing Information as defined in the associated requirements listed below.

Acceptance Criteria:
Acceptance criteria are defined as successful completion of the steps identified below with results matching the description of Expected Results.
