Validation Protocol

Test Description:
The purpose of this protocol is to demonstrate with ZemPro that the Dose Initiations Calculator prevents results calculations displaying error messages when the User enters values out of range or incorrect format. The User will be directed to the Contraindications, Warnings, Clinical Studies, Dosage and Administration sections within the US Prescribing Information relative to received warning.

Acceptance Criteria:
Acceptance criteria are defined as successful completion of the steps identified below with results matching the description of Expected Results.
