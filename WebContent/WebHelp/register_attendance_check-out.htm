Register Attendance Check-out

The resource can register their attendance check-out for the shift or the test factory admin can register the attendance check-out. By default, the actual check-out time of the shift is registered as shift check-out time of the resource. In the absence of the actual check-out time of the shift, the check-out time of the work shift set by the test factory manager is registered as the check-out time of the resource.

Prerequisites

Make sure that the resource manager registers the check-in time for the shift. You can register the check-out only after the check-in time for the shift is registered.

Register check-out time

To register the check-out time of a resource, follow these steps:

  1. Click My Tasks > My Attendance.
  2. In the Date field, select a date range, and in the Shift drop-down list, select the shift for which the attendance check-out time to be entered for.
  3. In the Check-Out Time field corresponding to a work date, select the date and time to mark the check-out time for that shift.

—OR—

To register/update the check-out time of a resource in the admin role, follow these steps:

  1. Click My Tasks > Resources Attendance.
  2. On the Resource Pool pane, select Test factory center > resource pool.
  3. On the Fill Resource Attendance pane, in the Pick a Date field, select a date, and select a shift.
  4. In the Check-Out Time field, enter the check-out time corresponding to the user ID. If required, enter the check-in time first, and then enter the check-out time.
