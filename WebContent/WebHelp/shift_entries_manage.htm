Schedule Actual Shift

Overview

This page allows the test manager to set the actual shift starting and ending time. Based on the actual shift starting and ending time, the check-in and check-out time of the resources will be updated automatically. The test manager can manually change the shift check-in time of the resource if required. For more information about changing the shift check-in time of a resource, see Register attendance check-in for a resource.

 

To access this page, login as Test Manager, and click My Tasks > Schedule Actual Shift.

Fields

Field

Description

Pick a Date

Enter the date for which the actual shift is scheduled for.

Start Time

Click this icon () to select the actual shift starting time.

Note: By default, the work shift starting time is displayed in this field. For more information about the work shift, see Test Factory Setup.

End Time

Click this icon () to select the actual shift ending time.

Note: By default, the work shift ending time is displayed in this field. Set the actual shift starting time first, then set the actual shift ending time.

Remarks

Enter the remarks or reasons if any.
