Assign User and User Roles for Product

The test factory manager can assign the user role for a specific resource, product team users, and core resources for the product. These resources are available only for this product, and they can perform the corresponding tasks based on the specified roles.

 

Assign user role for the product

You can assign a user role for a resource. The resource can act as the assigned role only for that product. For example, a resource say RM can be a Test Manager for a test factory, when you assign the Test Lead role  for a product. Then he can act as Test Lead for the product alone.

 

To assign a role for a user, follow these steps:

  1. Click Setup > Product Management, and then select Test Factory > Product.
  2. On the Products tab, click the User Role By Product () icon. As a result, the Add/Edit User Role for Product pane is displayed.
  3. Click the Add button, and enter the user name, and role type.
  4. Click the Save button to assign the user role for the resource.

Assign core resource of the product

You can assign a core resource for a product such that the resource is available only for the product.

To assign the product team user for the product, follow these steps:

  1. Click Setup > Product Management, and then select Test Factory > Product.
  2. On the Testing Team tab, on the Add/Edit Product Team Resources pane, click the Add button, and select the user name, user role, and from and to dates. The selected user will be available only for this product till the to date.
  3. Click the Save button to assign the resource for the product.

 

To assign the flex testing team user for the product, follow these steps:

  1. Click Setup > Product Management, and then select Test Factory > Product.
  2. On the Testing Team tab, on the Add/Edit Product Core Resources pane, click the Add button, and select the resource pool ID, user name, user role, and from and to dates. The selected user will be available only for this product till the to date.
  3. Click the Save button to assign the resource for the product.
  4. The Test Factory Core Resources pane displays the core resources available for the Test Factory. To include core resource for the test factory, contact Admin.
