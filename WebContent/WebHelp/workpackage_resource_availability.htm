Resource Demand/Availability

Overview

This page displays the demand and availability of resources for a product/work package on a specific day and shift.

 

To access this page, login as Test Manager, and click Resource > Resource Demand/Availability.

 

Fields

Field

Description

Demand drop-down list

Select one of the following options to display the status of the resource:

    • Demand

    • Availability

    • Reserved

    • Shortage

Shift

Displays available shifts for the Test Factory.
