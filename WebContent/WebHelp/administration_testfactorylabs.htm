Test Centers

Overview

This page allows you to create a new test factory, a resource pool, and assign core resources and user roles to the test factory. For more information about setting up a test factory, see Manage Test Factory, and for more information about creating a resource pool, see Resource Pool.

 

To access this page, login as admin, and click Admin > Manage Test Factory Labs.

The Test Centers pane displays the following fields:

Fields

Field

Description

Test Center

Name of the test center.

Display Name

Display name of the test center.

City

The city where the test center is located.

State

The state where the test center is situated.

Country

The country where the test center is in.

Status

Displays whether the test center is active or inactive.

 

Buttons

Field

Description

Add/Edit Test Factory ()

Click this button to add or edit a test factory.

Add/Edit Test Engagement ()

Click this button to add or edit a test engagement.

Resource Pools ()

Click this button to add or edit a resource pool.

Description

Add/Edit Test Factory Managers

Click this button to add a test factory manger. Only these test factory manager can create work shift and manage the resources for this test factory.

Click this button to add core resources for the test factory. These core resources are available only for this test factory.
