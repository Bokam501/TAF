Customer Accounts

Overview

The page displays list of available customers with their reference ID's. These customer IDs' are then mapped to products.

To access this page, login as Test factory manager or Admin, and click Setup > Customer Management.

Fields

Field

Description

Name

Customer name or reference name for the customer account.

Description

Description of the customer account.

Reference ID

An unique number assigned to the customer account.

Status

This checkbox displays whether the customer account is active or inactive.

Button

Field

Description

Add

Click this button to create a customer account.
