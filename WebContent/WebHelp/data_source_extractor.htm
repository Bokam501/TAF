Data Extractor

Overview

The Data extractor page is used to extractor data from various source files such as WATS, TASK, Effort sheet and so on. The extracted data is used to generate SLA metrics and charts used for tracking various metric. You can schedule the data execution periodically or manually trigger the task.

To access this page, click Setup > Data Extractor.

Fields

Field

Description

Name

Name of the data extractor.

Customer

Name  of the customer account.

Product

Product name for which the data to be mapped.

Competency

Competency of the product.

Type

Select one of the following file types:

  1. TASK
  2. WATS
  3. Query Log
  4. Effort
  5. Review Record
  6. Utilization
  7. Training Type

Location

File location of the source data.

Start Date

Starting date of the schedule, from which the data extraction to be triggered, periodically.

End Date

Ending date of the schedule, from which the data extraction to be stopped.

Last Executed

Displays the latest data extraction date.

Next Execution

Displays the next data extraction date, which is scheduled.

 

Buttons/Areas

Field

Description

Schedule ()

Click this icon to set up the data extraction schedule using cron schedule expressions. The Cron expression is a string which triggers the scheduler to execute the test run plans. For more details, Cron Expression.

Execute ()

Click this icon to trigger the data extraction.
