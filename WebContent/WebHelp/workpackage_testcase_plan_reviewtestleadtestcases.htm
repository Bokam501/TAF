Review Test Case Execution

Overview

This page allows you to review and approve the test cases executed by test lead and tester. You can view the test cases, resources who executed the test case, actual execution date, results of the test case, evidence if any, and defect details.

 

To access this page, click Review > Review Test Case Executions.

 

Fields

Field

Description

Execution Status

Select one of the following status:

  • Assigned - the test case is assigned to the resource
  • Completed - the test cases is executed by the resource
  • Not Started - the test case execution is not yet started

Reviewed

Select this checkbox to mark the test case as reviewed.

Approved

Select this checkbox to mark the test case as approved. This status implies that the test case is reviewed and completed.
