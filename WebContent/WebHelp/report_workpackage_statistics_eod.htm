Work Package Statistics

Overview

This page displays the list of work package executed for the select shift on a specific date. You can enter the performance rating to a user or approve the daily performance rating given to a user.

To access this page, click Reports > Work Package Statistics.

Fields

Field

Description

Date

Date on which the user executed the test case.

Work Package

Name of the work package executed.

Shift

Shift on which the user executed the test cases.

Test Lead

Name of the test lead who provided the performance rating for the user.

Tester

Name of the tester to whom the daily performance rating is provided for.

Test Cases

Total number of test cases executed in the selected date and shift for a work package.

Total Bugs

Total number of bugs logged by the user for the work package.

Hours Today

Total number of hours the user worked/taken to completed the test cases.

Ratings out of 5

Daily performance rating provided for the user by the Test Lead or Test Manager.

Rating comments

Rating comments provided by the Test Lead or Test Manager.

Rated by

Name of the person who provided the ratings.

Rated On

Date on which the daily performance rating is provided.

Approved

Select this checkbox to approve the rating provided.

 

Buttons/Areas

Field

Description

Export to Excel

Click this button to export the test case execution results to an excel sheet.
