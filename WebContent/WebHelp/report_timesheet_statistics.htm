Timesheet Statistics Report

Overview

This page displays the overall timesheet statistics report for the test factory for the selected date range. The page lists the details about shift, tester, shift attendance, project, resource activity, billing data of a tester, and cumulative billing effort.

 

To access this page, click Reports > Timesheet Statistics.

 

Fields

Field

Description

Date

Select a date to generate the timesheet statistics report.

Date Details

Displays date, week start and end dates, and which part of the month.

Shift Details

Displays the actual shift, start and end time of the shift, and duration of the shift.

Tester Details

Displays the tester name, test lead for the tester, vendor name of the tester, and other details.

Shift Attendance Details

Check-In and Check-Out timing of the tester, total duration worked, and shift status whether the tester came on time or delayed.

Project Details

Displays the Factory the tester belongs to, project/product, cost center, and work package name.

Resource Activity

Displays the role played by the tester, Timesheet hours, and Timesheet activities such as Break and Lunch timings.

Billing Data

Displays the total net hours the tester worked, regular work effort, and overtime details.

Cumulative Billing Effort

Displays shift wise timing, day wise timing, and week wise timing details.

 

Buttons/Areas

Field

Description

Generate Report

Click this report to generate the timesheet statistics for the selected date range.
