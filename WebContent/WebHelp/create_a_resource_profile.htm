Create a Resource Profile

The user such as Admin or Resource manager can create user profile for the resource such Test Lead, Tester, or Test Manager. The resource may belong to a test factory or test engagement. The resource who belongs to a test factory model are grouped in the resource pool based on the skill level, location, or other parameters of the resources. The resource manager can create resource profiles for the test factory that he is mapped.

Create a resource profile for a resource pool

To create a resource for a resource pool using the Admin role, follow these steps:

 

  1. Click Admin > Users > Manage Flex Users.
  2. On the Manage Users pane, select Test Factory Lab > Resource Pool. As a result, the User pane displays the list of existing users.
  3. Optional: On the User pane, click the Choose File button to import the existing user details in the excel format.
  4. On the User pane, click the Add button, and enter details of the resource such as First Name, Last Name, Middle Name, Login ID, Password, and Display Name.
  5. By default, a unique user Login ID is created for the user, which can be changed as required.
  6. In the User Type drop-down list, select the FTE, Contract Resource, Flexi or Customer option.
  7. In the User Role drop-down list, select the Admin, Engagement Manager, Test Manager, Test Lead, Tester, Center Manager, or Resource Manager option.
  8. In the Resource Pool drop-down list, select the corresponding resource pool for which the resource is allocated, and in the Vendor drop-down list, select the corresponding vendor account ID.
  9. Click the Save button to create the resource and assign him to the corresponding resource pool.
  10. If required, clear the Status checkbox to mark the resource account is inactive.
  11. Optional, select the All, Active or Inactive option in the Status drop-down list to filter the resource account in the selected status.
  12. If required, click the Delete button to delete the resource account.

 

—OR—

 

To create a resource for a resource pool using the Resource Manager role, follow these steps:

  1. Click Resource Management Pool > Manage Resource Pool Resources. As a result, the Manage Resource Pool Resource page displays Manage Users and User panes.
  2. On the Manage Users pane, select Test Factory Lab > Resource Pool. As a result, the User pane displays the list of existing users for the Resource pool.
  3. Repeat steps 3 through 12 from the above task.

Create a resource profile for regular users

To create a resource for a resource pool using the Admin role, follow these steps:

 

  1. Click Admin > Users > Manage Regular Users.
  2. Optional: On the Regular Users tab, click the Choose File button to import the existing user details in the excel format.
  3. On the User pane, click the Add button, and enter details of the resource such as First Name, Last Name, Middle Name, Login ID, Password, and Display Name.
  4. By default, a unique user Login ID is created for the user, which can be changed as required.
  5. In the User Type drop-down list, select the FTE, or Contract option.
  6. In the User Role drop-down list, select the Engagement Manager, Test Manager, Test Lead, Tester, Center Manager, or Resource Manager option.
  7. Click the Save button to create the resource profile.
  8. Click the Add/Edit User Customer Account(s) () icon.
  9. Click the Add button to add a customer account for whom the resource will work for the specified period.
  10. If required, clear the Status checkbox to mark the resource account is inactive.
  11. Optional: Select the All, Active or Inactive option in the Status drop-down list to filter the resource account in the selected status.
  12. If required, click the Delete button to delete the resource account.

 

—OR—

 

To create a resource for a resource pool using the Resource Manager role, follow these steps:

  1. Click Resource Management Pool > Manage Resource Pool Resources. As a result, the Manage Resource Pool Resource page displays Manage Users and User panes.
  2. On the Manage Users pane, select Test Factory Lab > Resource Pool. As a result, the User pane displays the list of existing users for the Resource pool.
  3. Repeat steps 3 through 12 from the above task.
