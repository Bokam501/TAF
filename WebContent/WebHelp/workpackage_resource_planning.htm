Resource Planning

Overview

The Resource Planning page allows you to allocate resources for a work package based on the demand raised for the work package on a specific date.

 

To access this page, click Resource > Resource Planning.

 

Fields

Field

Description

Pick a Date

Select a date for which resources are allocated to the work package.

Product

Displays the product name.

Work Package

Displays the work package ID.

Reservation Date

Date on which the resource is demanded for.

Shift

Shift for which the resource is demanded for.

Demand

Click this link to view the resource demand along with the roles.

Booked Resources

Click this link to view the resources booked for the work package.

Reserved

Click this link to view the resources reserved for the work package.

Gap

Click this link to view the resource demand gap for the work package.
