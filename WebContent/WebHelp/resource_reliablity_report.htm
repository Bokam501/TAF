Resource Reliability

Overview

This page displays the list of resources with their reliability result over the execution of the allocated work package.

 

To access this page, login as Test Manager, and click Reports > Resource Reliability.

Fields

Field

Description

Pick Date Range

Select from and to date to list the available resources with reliability result over the work packages.

Resource Pool

Name of the resource pool the user belongs to.

User

Name of the user.

Role

Role performed to completed the allocated task.

Vendor

Vendor name to whom the resource belongs to.

Total Bookings

Click this link to display the total booking summary of the user.

Show Up[%]

Click this link to display the total number shifts the user attended/worked.

On Time[%]

Click this link to display the total number days the user attended the shift on time.
