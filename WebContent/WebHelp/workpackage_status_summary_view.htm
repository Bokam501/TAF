Monitor Work Package

Overview

This page displays the list of work packages available for a product. The work packages are either completed or in progress.

 

To access this page, click Plan > Test > Monitor Work Package.

Fields

Field

Description

Work Package Name

Name of the work package.

Selected Testbeds

Testing test beds selected for the work package.

Selected Test Cases

Test cases selected for the work package.

Total Test Cases for Execution

Total number of test cases available for execution in the selected work package.

Completed Test Cases

Total number of test cases executed in the work package.

Not Completed Test Cases

Total number of test cases to be executed in the work package.
