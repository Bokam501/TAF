New Test Factory and Product Setup

This workflow diagram details different stages involved in creating a new test factory and a new product.

 

 

Outline of workflow

  • Test Factory Manager creates a test factory, and sets up work shifts for the test factory
  • Resource Manager maps the test factory with a resource pool
    • resource pool includes resources with the following roles test manager, test lead, and tester
  • Test manager
    • creates a product, version, and build
    • maps test case, test suites, test environments, and decoupling category
    • allocates core resources to the product
