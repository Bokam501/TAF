User creation

This workflow diagram details different stages involved in creating a user profile for the resource in a new test factory.

 

 

Outline of workflow

  • Administrator creates a vendor account, a resource pool, and resource profiles for the resource pool with the vendor account
  • Resource manager creates user profiles for tester, test lead, and test manager, and map the resource pool to the test factory.
