Work Package - Demand Projection

Overview

This page displays the resource demand projection for executing the test cases in a work package. This page also displays the Test cases planned for this week, and Resource demand for executing test cases panes. The test manager can view the resource demand from the Resource demand for executing test cases pane, and plan work package execution for the week.

To access this page, click Plan > Work Package - Demand Projection.
