Review executed test cases

The test manager or test lead can review the test case executed by the tester and test leads, and approve or reject them based on the review.

Review executed test cases

To review the completed test cases in a work package, follow these steps:

  1. Click My Tasks > Review > Review Test Case Executions.

  2. On the Review Test Case Execution page, in the Work Packages pane, select Product > Build > Work package.

  3. Select the Reviewed and Approved check boxes to confirm the review and approval status of the test cases.
