Analyze Defect

Overview

The Analyze Defect page displays the list of available defects based on the selected date and work package. Also, this page allows to upload the bugnizer data and compare with the defects. Based on the analysis, you can manually include the Issue ID to the defect.

 

To access this page, login as Test Manager, and click My Activities > Review > Analyze Defects.

 

Fields

Field

Description

Date Filter

Select the from date and to date to view the list of logged bugs and defects for the selected work package in the Defect Details table.

Issue ID Exist

Select one of the following options:

  • All - To list all the available defects for the selected work package
  • Yes -To list all the defects with Issue ID
  • No - To list all the defects without Issue ID

Analyzed

Select one of the following options:

  • All - To list all the available defects
  • Yes -To list all the defects which are analyzed
  • No - To list all the defects which are no analyzed

Status

Select an option to display the corresponding list of defects based on its defect status.

Issue ID

Enter the issue ID based on the existing defect item in the bugnizer.

 

Buttons

Field

Description

Import Defect

Click this button to upload the Bugnizer data from an excel sheet.

Bugnizer Data

Click this button to view the list of uploaded Bugnizer data.

Analyze ()

Click this button to view the Analyze Defect page. This page displays the defect details and Bugnizer defect data. Bugnizer defect data will be listed based on the feature name and type of release.
