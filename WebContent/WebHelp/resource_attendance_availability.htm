Fill Resource Attendance

Overview

This page allows you to fill the shift check-in and check-out time of the resources, if the test manager or the resource does not filled the shift check-in or check-out.

 

To access this page, login as Admin, and click My Task > Resources Attendance.

Filters

Field

Description

Pick a Date

Select a date for which the resource attendance is filled for.

Shift

Select one of the following to list the resource attendance:

  • All

  • Morning

  • Night

  • Graveyard

 

Fields

Field

Description

Resource Pool

Name of the resource pool, the resource belongs to.

User

Name of the resource whose attendance check-in and check-out timings are marked for.

Role

Role of the resource.

Attended

Select this checkbox to register the attendance for the resource. By default, the current system time will be applied in the Check-in Time field.

Check-in Time

Select the check-in time of the resource.

Note: This field update with the current system time, if you select the Attended checkbox.

Check-out Time

Select the check-out time of the resource.
