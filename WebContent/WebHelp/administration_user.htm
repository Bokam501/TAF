Resource

This page allows a user to create a resource account. The resource manager creates the user profile for resources with the profile details such as login ID, email ID, user type and role, resource pool ID, and vendor account.

 

To open the resource page, follow the step:

If the user is admin, click Admin > Manage Application Users.

— OR —

If the user is resource manager, click the Resource Pool Management > Manage Resource Pool Resources.

 

Fields

Field Label Description
First Name Enter the first name of the resource.
Last Name Enter the last name of the resource.
Middle Name Enter the middle name of the resource.
Login ID By default, an unique login ID is created based on the first name and last name of the resource. The admin or resource manager can change the Login ID.
 
Note: If you are manually changing the login ID, make sure the login ID is unique.
Password Enter the password for the resource login.
 
Note: The password length must be minimum 6 characters, and must include number, upper and lower cases, and special characters.
Display Name Enter the display name of the resource.
Email ID Enter the e-mail address of the resource. After the resource credentials are created, an e-mail is send to the user along with the resource credentials.
User Type Select one of the following options:
  • FTE - This resource is working as Full-time equivalent (FTE) employee.
  • Contract Resource - This resource is working under contract for the test factory.
  • Flexi - This resource is working in flexi shifts.
User Role Select one of the following roles:
  • Admin
  • Resource Manager
  • Test Factory Manager
  • Test Manager
  • Test Lead
  • Tester
Languages Default language.
Resource Pool Select a resource pool ID for which the resource belongs.
Vendor Select a vendor account for which the resource is working for.
