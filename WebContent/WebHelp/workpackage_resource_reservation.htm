Reserve Resources For Work Package

Overview

This page allows you to reserve resources for different work packages for the selected date and shift. Drag and drop the resources from the Available Resources list to the Work package list to reserve the resource to the work package.

 

To access this page, click Resource > Resource Reservation.

Fields

Field

Description

Date

Select a date for which resources are reserved for.

Shift

Select a work shift to display the available resources for that shift.

Booked/Booked or Available/All

Click one of the options to display the list of booked, booked or available, or all resources for the selected date and shift.
