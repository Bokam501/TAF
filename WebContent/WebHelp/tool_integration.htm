Tool Integration

This feature facilitates the integration of external Test and Defect Management Systems with iLCM. It records the mapping details necessary for the integration. It allows mapping each product available in the iLCM with multiple instances of Test and Defect Management Systems.

 

Prerequisites

Make sure the following things are in place:

  1. Valid user credentials for the Test and Defect Management Systems.
  2. Entity value is available in the Test and Defect Management Systems.

Test Management System

To configure a Test Management System, follow these steps:

  1. Click Setup > Product Management. On the Products pane, select the Engagement > Product.
  2. On the Manage Products page, click the Tool Integration tab.
  3. On the Test Management System tab, click the Add button. As a result, the Add page is displayed.
  4. In the Title field, enter the test management system name i.e. HPQC, and description, and enter the cron expression. The Cron expression is a string which triggers the scheduler to execute the test run plans. For more details, Cron Expression.
  5. In the Test System drop-down list, select the test management system as HPQC, and in the System Type drop-down list, select the either 32-bit or 64-bit OS.
  6. Is Primary check box to set the configured management system as primary system for test cases import.
  7. In the Version field, enter the test management system version number, and in the Connection Uri field, enter the url of the test management system say “http://10.108.57.17:8080/qcbin”.
  8. Enter the username, password, and domain, which is the domain name of the management system, and then click the Save button. TestSetFolder Filter (ID), TestSetFolder Filter(NAME), TestCase Filter(ID), and Resource filter (ID) are place holder fields for future reference.

Before performing the uploading or downloading operation, you must establish the connection between the HPQC and iLCM. For example, the HPQC has a domain i.e. default, and a project i.e. product1. Add the entity details to import the test cases from the HPQC. To add the entity details, follow these steps:

  1. On the Test Management System tab, click the Test Management System Mapping () icon, and then on the Test Management System Mapping pane, click the Add button.
  2. In the Mapping Entity drop-down list, select one of the following options:
    • Product (Select this option to view the list of products available in the HPQC)
    • Product Version
    • Test Suite
  3. In the Entity Name drop-down list, select the entity name i.e. Product1. The test case for the selected product name will be imported from the test management system.
  4. In the Mapping Value field, enter the mapping value for the product name i.e. Product. Here the Mapping value indicates the project name in HPQC system.

Defect Management System

To configure a Defect Management System, follow these steps:

  1. Click Setup > Product Management. On the Products pane, select the Engagement > Product.
  2. On the Manage Products page, click the Tool Integration tab.
  3. On the Defect Management System tab, click the Add button. As a result, the Add page is displayed.
  4. In the Title field, enter the defect management system name, and in the Description field, enter the description for the defect management system.
  5. In the Test System drop-down list, select the defect management system like Jira.
  6. In the Version field, enter the defect management system version number, and in the Connection Uri field, enter the url of the defect management system.
  7. Enter the username and password, and then click the Save button.

To add the entity, follow these steps:

  1. On the Defect Management System tab, click the Defect Management System Mapping () icon, and then on the Defect Management System Mapping pane, click the Add button.
  2. In the Mapping Entity drop-down list, select one of the following options:
    • Product
    • Product Version
  3. In the Entity Name drop-down list, select the product name. The defect results of the selected product will be imported to the defect management system.
  4. In the Mapping Value field, enter the mapping value for the product name.
