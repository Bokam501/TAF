Approve Timesheets

Overview

This page allows you to approve timesheet activities of resources.

 

To access this page, login as Test Manager, and click My Tasks > Approve Timesheets.

Fields

Field

Description

Shift

Select the actual shift to make correction.

Hours

Select the actual working hours spent for the work activity.

Mins

Select the actual working minutes spent for the work activity.

Activity Type

Select the work activity type.

Approve Activity

Select this checkbox to approve the timesheet activity.

Approver Comments

Enter comments to the resource, if any.

 

Filter

Field

Description

Timesheet Approval Status

Select one of the following status options:

  • All - Select this option to list all the timesheet activities.

  • Approved - Select this option to filter the approved timesheet activity.

  • Unapproved - Select this option to filter the unapproved timesheet activity.
