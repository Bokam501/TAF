Status Management

Overview

Status category defines the number of status involved in an activity work flow, where the status can be mapped to one or more products. To access this page, login as admin, and click Setup > Status Management.

  • Status Summary
  • Primary Status
  • Secondary Status

Status Summary

The status summary tab displays status creation details, primary status and secondary status along with the status counts.
