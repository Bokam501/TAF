Activity

Overview

This page allows you to create an activity for the activity work package with the planned starting and ending dates.

 

To access this page, click Plan > Activity Management > Activity.

Fields

Field

Description

Activity Name

Enter the unique activity name.

Work Package

Displays the activity work package name.

Requirement

Select the requirement (Feature) from the drop-down list.

Activity Type

Select the activity type for the grouping the activity.

Tracker Number

Enter the tracker number, if required.

Category

Select one of the following category: manual, automated or both.

Assignee

Select the assignee name for the activity.

Reviewer

Select the reviewer name to review the activity executed by the assignee.

Priority

Enter the priority of the activity.

Status

Select the execution status of the activity.

Active

Select this check box to change the activity state to active or inactive.

Planned Start Date

Select the planned starting date to execute the activity.

Planned End Date

Select the planned ending date to complete the activity.

Actual Start Date

Select the actual starting date of the activity.

Actual End Date

Select the actual end date of the activity.

Planned Activity Size

Enter the planned activity size. Activity size might be number of same kind of executions to be performed.

Actual Activity Size

Enter the actual activity size.

 

Buttons

Field

Description

Add

Click Add to the activities.

Change Request

Click this icon and add the change requests received.

Change Request Mapping

Click this icon, and drag and drop the available change request.

Clarifications

Click this button to display the list of available clarifications raised for the work package along with the Title, Priority, Status, and Owner of the activity.

Environment Mapping

Click this icon, and drag and drop the available environment combination for the activity.
