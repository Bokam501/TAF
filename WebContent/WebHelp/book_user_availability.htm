Book Available Users

The resource manager can view the resource in demand for the shift. Based on the demand, the resource manager books the resource who are marked their availability. The resource manager can communicate to on demand resources but not available for the shift, and confirms their availability to book them for the shift.

Book user availability for a work shift

To book a resource for a shift, follow these steps:

  1. Click Resource Management > Resource Fulfilment.
  2. On the Resource Pool pane, select a resource pool, and on the Resource Management pane, click the Book Resources tab.
  3. In the Pick a Date field, select the date for which the resource availability is booked for, and select the work shift from the shift drop-down list. As a result, the Resource Available pane displays the list of users.
  4. Select the Book - Shift checkbox corresponding to resources for booking the resource for the work shift.
    Note: If the resource does not mark his availability for the work shift, the resource manager can mark his availability for the work shift, and then book his availability.
