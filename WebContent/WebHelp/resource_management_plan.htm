Resource Fulfilment

Overview

The Resource Fulfilment page allows you to view the resource demand summary for a product, to book resources from a resource pool for a specific shift of a date, and to resolve resource reservation conflicts.

To access this page, login as Resource manager, and click Resource Management > Resource Fulfilment. Click tab links below for field level information.

  • Resource Demand Summary
  • Available Resources
  • Book Resources
  • Reservation Conflicts

Resource Demand Summary

On the Resource Pool pane, select the corresponding test factory, and then select the Resource Demand Summary tab.

Fields

Field

Description

Demand

Select one of the following from the drop-down list:

  • Demand - Displays the resource demand in the calendar.

  • Availability - Displays total number of available resources in the calendar even when the available resources are reserved.

  • Reserved - Displays the reserved resources for the date in the calendar.

  • Shortage - Displays difference of the resource demand count and resource reserved count.

Shift

Select the shift from the drop-down list to filter and displays the resource demand, availability, reserved, or shortage details in the calendar.

 

Button

Field

Description

Previous/Next

Click this button to navigate to the previous/next month in the calendar.

 

Available Resources

On the Resource Pool pane, select the corresponding resource pool, and then select the Available Resources tab. This tab displays available resources in resource pools for the selected date and shift. The resource manager confirms the resource availability for the selected date and shift.

 

Fields

Field

Description

Pick a Date

Select a date to view the list of the resource to be booked for the selected date.

Shift drop-down

Select a shift from the drop-down list to filter the resources based on the selected shift.

Remaining Timesheet Hour

Displays the remaining timesheet hour the resource can be work for the selected week.

Remaining Booked Hour

Displays the remaining hour the resource can be booked for.

Availability - Shift

Select this checkbox to confirm the availability of the resource for the selected date and shift.

Note: This checkbox is selected if the resource marks their availability for the selected date and shift. For more information, see Mark user availability.

Book - Shift

Select this checkbox to book the resource for the selected date and shift.

Summary

Click this button to display the Resource Available Summary page. This page lists the resources demand, availability, booked and shortage details.

Resource Conflicts

On the Resource Pool pane, select a resource pool, and then select the Reservation Conflicts tab. This tab displays list of resource reservation conflicts. You can select a date range to view the resource reservation conflicts.

Fields

Field

Description

Calendar

Click this icon to select the from and to date, and click the Apply button to list the resource reservation conflicts.

Conflicting Reserved Resource

Click this link to view the List of Conflicting Reserved Resources page. Select the resource checkbox, and click the Confirm Reservation(s) button to confirm the reservation of the resource for the shift.
