Resource Tracking

Overview

This page allows you to register the attendance and shift check-in time of a resource, and to monitor the resource reliability for a specific period.

To access this page, login as resource manager, and click Resource Management > Resource Tracking.

  • Resource Attendance
  • Resource Reliability

Resource Attendance

This tab allows to filter resources based on the selected date and shift, and register the resource presence.

Fields

Field

Description

Resource Pool

Name of the resource pool, the resource belongs to.

User

User name of the resource for which the attendance is registered for.

Role

Role of the resource.

Skills

Skills expertized by the resources.

Vendor

Vendor name for whom the resource working for.

Booked

Booking status of the resource for the day/shift.

Attended

Select this checkbox to mark the attendance for the day.

Note: If you select this checkbox, the Check-in Time field will be updated with the current time.

Check-in Time

Select the check-in time of the resource for the shift/day.

Note: if you select the check-in time of the resource, the Attended checkbox is selected automatically.

 

Buttons/Filters

Field

Description

Summary

Click this button to view the resource availability summary of the selected date.

Pick a Date

Select a date to filter the resource available for the day.

Shift

Select a shift to filter the resource available for the shift on the selected date.

 

Resource Reliability

This tab allows you to view the resource reliability for a specific period of time.

Fields

Field

Description

Date

Select the from and to date to view the summary of the resource reliability.

Total Bookings

Total number of bookings made for the resource.

Show Up [%]

Percentage of presence over the bookings count. For example, if the user is booked for 4 shifts, and he is present for only 3 shifts, then the show up percentage is 75%.

On Time [%]

Percentage of number of on-time presence over the actual shift check-in time. For example, if the actual shift starts by 17 hrs, and the user check-in by 18 hrs on the first day, and 17 hrs on the second day, then the on-time percentage is 50%.

 

Links

Field

Description

Show Up [%]

Click this link to view the Show Up Summary page. This page displays the list of resources who are available for a specific work package, shift on which the work package is executed, reservation date, and name of the person who reserved the resource.

On Time [%]

Click this link to view the On Time Summary page. This page displays the list of resources who are available for a specific work package, shift on which the work package is executed, reservation date, and name of the person who reserved the resource.
