Regular Users

Overview

This page allows you to create and manage the user account of the full time employees/contract employees.

To access this page, login as Admin, and click Admin > Manage Product Team Users.

Fields

Field

Description

First/Middle/Last Name

First, Middle, and Last names of the employee.

Note: First and Last names of the employee is mandatory.

Login ID

Unique login identifier.

Display Name

Display name of the employee.

Email ID

Enter the e-mail address of the resource. After the resource credentials are created, an e-mail is send to the user along with the resource credentials.

Contact Number

Contact number of the employee.

Note: Format is country code, network prefix, and number. For eg. USA (Country Code = 1, network prefix = 428, number = 1234567).

User Type

Select one of the following options:

  • FTE - This resource is working as Full-time equivalent (FTE) employee.

  • Contract Resource - This resource is working under contract for the test factory.

User Role

Select one of the following roles:

  • Admin

  • Resource Manager

  • Test Factory Manager

  • Test Manager

  • Test Lead

  • Tester

Languages

Default language.

Status

Select this checkbox to ensure that the user profile is active.

Skill Name

Skills know to the user.

Authentication Type

Based on the selected option, the user credentials will be verified.

Select one of the following options:

  • Local - the user is local user.
  • Enterprise - the user is either customer or vendor user.

Authentication Mode

Select one of the following options:

  • Customer - the user is customer user.
  • Vendor - the user is vendor user.

 

 

Buttons

Field

Description

Reset Password

Click this button to reset the password to default password (IDpa@123).

Note: Please advice the user to change the password after receiving the default password.

Add/Edit Customer Account

Click this button to add the customer account and time period to work for that customer product. Enter the customer account, code, email address, and from and to date.

Delete

Click this button to delete the user account.
