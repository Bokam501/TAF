Environment Group

Overview

The Environment Group page allows to create testing environments and testing environment groups. The testing environment group is collection of testing environments. You can define the testing environment in which the product is to tested.

To access this page, login as admin, and click Admin > Environment Group

Fields/Buttons/Areas

Field

Description

Environment Group

Displays the name of the testing environment group.

Description

Brief description of the environment group.

Display Name

Display name of the environment group.

Created Date

Date on which the environment group was created.

Modified Date

Date on which the environment group was modified.

Status

Select this checkbox to set the environment group status to active.

 

Buttons

Field

Description

Add

Click this button to add a new environment group.

Environment Category

Click this button to displays the Add/Edit Environment Category pane. The Add/Edit Environment Category pane displays a list of environment category.

Field

Description

Environment Category

Name of the environment category.

Description

Brief description about the environment category.

Status

Select this checkbox to set the environment category status to active.

Parent Environment Category

Displays the parent/environment group name.

Add Button

Click this button to add a new environment category.
