Approve Skills

Overview

This page allows you to approve the skills set of resources.

 

To access this page, click My Tasks > Approve Skills.

Fields

Field

Description

User Name

Name of the resource.

Skill

Name of the primary/secondary skill possessed by the resource.

Primary Skill

Select this checkbox if this is the primary skill of the resource.

Skill Level

Displays expertise skill level registered by the resource.

Approver

Name of the approver.

Approved Skill Level

Select one of the expertise skill level that the resource possess:

  • Novice

  • Advanced Beginner

  • Competent

  • Proficient

  • Expert

Approved

Select this checkbox to approve the expertise skill level of the resource.

From Date

Displays the validity from date of the skill.

To Date

Displays the validity to date of the skill.

 

Filters

Field

Description

Status

Select one of the following options from the drop-down list:

  • All - Displays approved and unapproved records of the skill level.

  • Approved - Displays approved records of the skill level.

  • Unapproved - Displays unapproved records of the skill level.

Skill Category

Select one of the following options from the drop-down list:

  • All - Displays primary and secondary skill level records.

  • Approved - Displays primary skill level records.

  • Unapproved - Displays unapproved skill level records.
