Create a New Product

The test factory manager can create a customer product, and versions and builds for the product.

Create a product

To create a product of the customer, follow these steps:

  1. Click Setup > Product Management. As a result, the Manage Product page is displayed.
  2. On the My Products pane, select the Engagement > Product, and then on the Products tab, click Add.
  3. In the Add page, select the engagement and customer account ID, and then enter the product name, description, project code for the product, and project name.
  4. Optional: Click Edit Competencies to map the competencies for which the product must be available.
    Note: This field is used for activity management for now, and doesn’t support test life cycle.
  5. Optional: Click Edit Workflow Status to map the process work flow for the product.
    Note: This field is used for activity management for now, and doesn’t support test life cycle.
  6. Click Save to create the product.

Create a product version

To create a product version, follow these steps:

  1. Click Setup > Product Management, and then select Test Factory > Product.
  2. On the Products tab, click the Add/Edit Product Version() icon. As a result, the Add/Edit Product Version(s) pane is displayed.
  3. Click Add, and enter the product version, description, and release date.
  4. Click Save to create the product version.

Create a product build

To create a build version, follow these steps:

  1. Click Setup > Product Management, and then select Test Factory > Product.
  2. On the Products tab, click the Add/Edit Product Version() icon, and then click the Add/Edit Product Build() icon. As a result, the Add/Edit Product Build(s) pane is displayed.
  3. Click the Add button, and enter the build number, name, description, and build date.
  4. Click the clone build () icon to clone the existing build along with the work packages.

    1. Enter the new build number, name, and description.

    2. Select the build type and build date.

    3. Click Save to create a new build with the similar work package details that are available in the existing build. The work packages will be created in the planning status.

  5. Click this () icon to clone the existing build along with the work package.
  6. Click the Save button to create the build for the product version.
