Vendor Account

Overview

This page allows the user to create a vendor account. The user can set up vendor account details such as single point of contact person's name, e-mail address, and contact number. Resources are assigned with the vendor account, which facilitates the user to identify for which vendor the resource belongs to.

 

To access this page, login as Admin, and click Admin > Manage Vendors.

Fields

Field

Description

Company Name

Name of the vendor company.

Domain

Domain name of the vendor company.

For eg. @hcl.com

SPOC Name

Single point of contact for the vendor company.

Email Address

Email address of the contact person. For eg. name@gmail.com.

Contact Number

Contact number of the vendor company.

 

Note: Format is country code, network prefix, and number. For eg. USA (Country Code = 1, network prefix = 428, number = 1234567).

Status

Select this check box to change the vendor company to active status.

Field
