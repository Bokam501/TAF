Register Attendance Check-in for a Resource

The resource manager can register the attendance check-in for a resource. Based on the resource attendance billing hours of a resource is calculated.

 

Register the check-in time of resources

To register the check-on time of a resource, follow these steps:

  1. Click Resource Management > Resources Tracking.
  2. On the Resource Pool pane, select a Test Factory Laboratory > Resource Pool, and then on the Resource Attendance tab, select a date, and shift from the drop-down list. As a result, the list of resource is displayed.
  3. If require, select the Billing Mode checkbox, and in the Check-In Time field, select the check-in time. As a result, the Attended checkbox is selected.
  4. The resource can verify the check-in status in the My Attendance page.
