Dimension Management

Overview

Dimension management includes status category and competency management. You can create the status category and competency in this page.

To access this page, click Setup > Dimension Management.

Fields

Field

Description

Name

Name of the dimension such as competency or status category.

Description

Description of the dimension.

Type

Type of the dimension such as competency or status category.

Parent

Parent node for the dimension if specified.

Status

Select this check box to change the dimension status to active or inactive.

Owner

Displays the person who created this dimension.

 

Buttons

Field

Description

Add

Click this button to create a new dimension.
