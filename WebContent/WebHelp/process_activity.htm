Activity Task

Overview

This page allows you to create tasks for an activity with the planned starting and ending dates.

 

To access this page, click Plan > Activity Management > Activity Tasks.

Fields

Field

Description

Task Name

Enter the unique activity task name.

Task Type

Select the task type from the drop-down list.

Category

Select one of the following: Manual, Automated or Both.

Environment Combination

Select the environment combination mapped to the activity.

Assignee

Select the assignee name for the activity task.

Reviewer

Select the reviewer name to review the activity task executed by the assignee.

Priority

Enter the priority of the activity task.

Primary Status

Select the primary execution status of the activity task.

Secondary Status

Select the secondary execution status of the primary status.

Result

Select the task execution result.

Active

Select this check box to change the activity task state to active or inactive.

Planned Start Date

Select the planned starting date to execute the activity task.

Planned End Date

Select the planned ending date to complete the activity task.

Actual Start Date

Select the actual starting date of the activity task.

Actual End Date

Select the actual end date of the activity task.

Planned Activity Size

Enter the planned activity task size. Activity task size might be number of same kind of executions to be performed.

Actual Activity Size

Enter the actual activity task size.

Planned Effort

Enter the planned effort to be spend for the task.

Edit Status

Click this icon to change the status of the task and to enter the effort spent for the task.

Total Effort

Displays the overall effort spent for the task.

 

Buttons

Field

Description

Add

Click Add to the activity tasks.
