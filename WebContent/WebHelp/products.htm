Product Setup

A product is an application to be tested for bugs and accuracy of business needs in a test factory. Development stage of the product is marked by its version and build number. The version numbers are package with new features and bug fixes. The build numbers are packages with a minimal fixes for the bugs and defects. You can assign the test suites, test cases, de-coupling category, testing devices, features, and testing environment to the product. You can also define or set test suites, and test run plans to product versions.
