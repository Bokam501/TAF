View Resources Experience

Overview

This page displays the overall experience of a resource on the product. You can view the product name, version, resource name, assigned work package count, test cases executed count, reported and approved defects for the test cases, and average performance rating for the resource.

 

To access this page, login as test manager, and click Monitor > View Resource Experience.

 

Fields

Field

Description

Work Package Count

Click this link to view the list of work packages allocated to the resource.

Executed Test Case Count

Click this link to view the list of test cases executed by the resource.

Average Performance Rating

Click this link to view the list of work packages and their corresponding performance rating of the resource provided by the  test lead/test manager.
