Dashboard Tab Setup

Overview

The Dashboard Tab Setup page is used to set up the tabs for displaying the SLA metrics and dashboards.

 

To access this page, and click Dashboard > Dashboard Tab Setup.

Fields

Field

Description

Tab Name

Name of the tab to be displayed in the Home page.

Engagement

Displays the name of the engagement. The tab will be displayed only to the selected engagement.

Deployment

Place holder text. Can be used as identifier.

Order No

Tab's order number to be displayed in the home page.

Status

Clear this check box to disable the tab.

 

Buttons

Field

Description

Add

Click this button to add a new tab.

Role Based Tabs ()

Click this button to add charts and summary content to the tabs for a specific role.

Field

Description

User Role

User role for which the chart or summary page will be displayed.

URL

Role specific urls for the dashboards and summary content.
