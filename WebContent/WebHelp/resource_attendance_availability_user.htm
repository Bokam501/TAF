Resource Attendance

Overview

This page allows the resource to verify the check-in registered by the test manager for the selected date, and to register the shift check-out time for the select.

 

To access this page, click My Tasks > My Attendance.

Filters

Field

Description

Date

Select the from and to date to view the attendance check-in and check-out for the select time period.

Shift

Select a shift from the drop-down list to view the attendance check-ins for the selected shifts.

 

Fields

Field

Description

Resource Pool

Name of the resource pool, the resource belongs to.

User

Name of the resource.

Role

Role of the resource on the selected date.

Work Date

The actual work date for which the check-in records are displayed.

Shift Name

The actual shift for which the check-in records are displayed.

Check-in Time

Check-in time for the work date.
Note: You can't select/modify the check-in time of the work date. To change the check-in time of the work date, contact your resource manager.

Check-out Time

Select the shift check-out time for the displayed work date.
