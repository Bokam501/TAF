Set Up Activity Type and Weightage

In the Activity management, each activity is grouped under an activity type. Based on the historical value of the activity performed, each activity type is associated with weightage for allocation and metric purpose.

 

Prerequisites

Make sure customer account exists for creating the activity type.

Create Activity Type

For every customer account, we can associate a set of activity types and their corresponding weightage. To create activity type, follow these steps:

  1. Click Admin > Manage Customers > Manage Activity Type, and on the Customer pane, select a customer account.
  2. Click Add, and in the Add page, enter the name and description for the activity type.
  3. Select an activity group, and activity category.
  4. If required, enter the weightage of the activity type, and then click Save to create the activity type with the weightage.
