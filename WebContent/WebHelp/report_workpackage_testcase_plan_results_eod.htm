Work Package Test Case Results

Overview

This page displays the test results of the work package for a specific date and product.

 

To access this page, click Reports > Work Package Test Case Results.

Fields

Field

Description

Work Package

Displays the name of the work package executed.

Shift

Displays in which shift the test case was executed.

Tester

Displays the tester name who executed the test case.

Environment

Displays the name of the testing environment combination.

Test Case

Displays the name of the test case executed.

Result

Displays the execution result of the test case.

Defect ID

Displays the defect ID, if any.

 

Buttons/Areas

Field

Description

Export to Excel

Click this button to export the test case execution results to an excel sheet.
