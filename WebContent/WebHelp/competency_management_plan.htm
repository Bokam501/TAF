Competency Management

Overview

Competency management defines the sub division of the engagement, in which it can be mapped and management across multiple products.

To access this page, click Setup > Competency Management.

  • Competency Summary
  • Products
  • Team

Competency Summary

Fields

Field

Description

About Competency

This section displays the details about the competency.

Members

This section displays the total number of members available in the competency.

Products

This section displays the mapped member details for the products.

 

 

Products

This tab displays the products mapped to the competency. In the Product Management page, you can map the competency to a product.

 

Fields

Field

Description

Product Name

Name of the product which is mapped to the competency.

Begin Date

Select the beginning date from which the product is available in the competency.

End Date

Select the ending date for the product to be available in the competency.

Status

Displays the active or inactive product mapping status.

Mapped By

Displays the name of the person who mapped the product to the competency.

 

Buttons/Areas

Field

Description

Product Members

Click this icon and add the resource to the competency.

 

Team

In this tab, you can add the team members for the competency. Based on which you can add the product team members in the Products tab.

 

Fields

Field

Description

Member Name

Displays the name of the competency member.

Begin Date

Select the beginning date from which the team member is available for the competency.

End Date

Select the ending date for the team member to be available for the competency.

Status

Select the check box to change the team member status to active or inactive for the competency.

Mapped By

Displays the person's name who mapped the user to the competency.

 

Buttons/Areas

Field

Description

Add

Click Add to add the team member to the competency for a specific period of time.
