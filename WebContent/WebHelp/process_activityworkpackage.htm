Activity Work Package

Overview

This page allows you to create a new activity work package for a build with the planned starting and ending dates. Activity work packages are created to monitor the activities to be performed by the engineers.

 

To access this page, click Plan > Activity Management > Activity Work Package.

Fields

Field

Description

Build

Enter the build number for which this activity work package is created.

Work Package Name

Enter a unique activity work package name.

Owner

Enter the person who takes responsibility for this work package.

Priority

Enter the priority of the work package.

Status

Select the execution status of the work package.

Competency

Select a competency for which the work package belongs to.

Active

Select this check box to change the work package execution to active or inactive.

Planned Start Date

Select the planned starting date to execute the work package.

Planned End Date

Select the planned ending date to complete the work package.

Actual Start Date

Select the actual starting date of the work package.

 

Buttons

Field

Description

Clarifications

Click this button to display the list of available clarifications raised for the work package along with the Title, Priority, Status, and Owner of the activity.

Add

Click this button to create a new activity work package.
