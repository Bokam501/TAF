Test Automation Framework

Test automation framework is a service accelerator to simplify the complexity in test automation of web applications and mobile applications on real world devices across platforms.

Fields

Field

Description

Terminal Name

Displays the name of the terminal, in which the TAF.jar file is running on.

IP Address

Displays the IP address of the terminal machine.

MAC

Displays the MAC address of the terminal machine.

Platform

Displays the operating system/platform of the terminal machine. For example, Windows 7.

Version

Displays the TAF.jar version.

Architecture

Displays the architecture version of the terminal machine. For example, if the terminal is running in 32bit processor, the architecture is x86.

Server Name

Displays the server name, whose URL is specified in the TAF.conf file.

Server IP

Displays the server IP address.

 

After the terminal and server connection is established, the Device(s) Detected and Jobs table is updated with the corresponding details.

Jobs

Field

Description

Job

Displays the executed job ID.

Product

Displays the product name for which the job is running for.

Device

Displays the device on which the job is running.

Status

Displays the execution status of the job.

Test Steps

Displays the number of test steps being progressed by total number of test steps.

Time

Displays the total time taken to complete the job.

Abort

Click this button to abort the execution of job in progress.
