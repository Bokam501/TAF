Resource Shift Approvals

Overview

This page allows you to view and approve the shift timings registered by resources such as test lead and tester. The shift timings must be approved to generate the timesheet statistics of resources.

To access this page, login as test manager, and click My Tasks > Shift Approvals.

Fields

Field

Description

User

Name of the resource, who registered shift timings.

Shift

Actual shift worked by the resource.

Check-In/Check-Out

Actual shift check-in/check-out timing.

Remarks

Remarks entered for the shift timings.

Approved

Select this checkbox to approve shift timings registered by the resource.

Approver

Name of the approver who approved shift timings.

Approved Date

Date on which shift timings are approved.

Remarks

Remarks included by the approver.

 

Filters

Field

Description

Date

Select a date to filter the shift timing records to be approved.

Status

Select one of the following options from the drop-down list:

  • All - Displays approved and unapproved shift timing records of the skill level.

  • Approved - Displays approved shift timing records of the skill level.

  • Unapproved - Displays unapproved shift timing records of the skill level.

Shift

Select a corresponding actual shift to filter the shift timing records to be approved.
