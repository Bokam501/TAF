Test Factory Setup

Overview

This page allows the user to create a test factory, and configure work shift timing for the test factory. The test factory manager can create test factories for a test factory center, and define work shifts for the test factories. Also, the test factory manager can view resource pools assigned to the test factory, where the resource manager assigns the resource pools to the test factory. For more information about assigning resource pools to a test factory, see Manage resource pool.

To access this page, login as test factory manager, and click Setup > Test Factory Management. Click tab links below for field level information.

  • Work Shifts
  • Resource Pools
  • Test Factories

Work Shifts

On the Test Factory pane, select the corresponding test factory, and then select the Work Shifts tab.

Fields

Field

Description

Shift Name

Name of the work shift.

Display Label

Display name of the work shift. This name will be displayed in the shift check-in, work package allocations, and so on.

Description

Detailed description of the work shift.

Shift Type

Select one of the following work shift type:

  • Morning
  • Night
  • Graveyard

Start Time

Select Hours and Minutes for start time of the work shift.

End Time

Select Hours and Minutes for end time of the work shift.

 

Button

Field

Description

Add

Click this button to create a new work shift for the test factory.

 

Resource Pools

On the Test Factory pane, select the corresponding test factory, and then select the Resource Pools tab. This tab displays available resource pools for the selected test factory.

Note: For Test Engagement, the Resource Pools tab is disabled.

Fields

Field

Description

Resource Pool Name

The resource pool name assigned to the test factory.

Display Name

The display name of the resource pool name.

City

The city where resources belongs to.

State

The state where resources belongs to.

Country

The country where resources belongs to.

Test Factory Lab

The test factory center to which resource pools are mapped to.

Status

Displays the active or inactive status of the resource pool.

 

Test Factories

On the Test Factory pane, select the test factory center, and then select the Test Factories tab. This tab displays list of available test factories.

Fields

Field

Description

Test Factory Name

 Name of the test factory.

Display Name

The display name of the test factory name.

City

The city where test factory is installed.

State

The state where test factory is located.

Country

The country where test factory is situated.

Status

Select the checkbox to change the test factory to active status.

 

Button

Field

Description

Add

Click this button to create a test factory.
